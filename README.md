# Проекты и прототипы

Мультипроектная статическая площадка: каждый проект живёт в своей папке с собственным
`index.html` и открывается со стартовой страницы. Ничего не собирается и не
компилируется — это чистый HTML/CSS/JS, данные вшиты внутрь страниц.

## Структура

```
.
├── index.html                     # стартовая страница-меню (карточки проектов)
├── package.json                   # один скрипт: локальный статический сервер
├── roboforex-organic-analytics/
│   └── index.html                 # дашборд органического трафика RoboForex (данные внутри)
└── compound-calculator/
    ├── index.html                 # меню концептов калькулятора
    └── concepts/
        ├── concept-1-live-duel.html
        ├── concept-2-characters.html
        ├── concept-3-do-nothing.html
        ├── concept-4-composite.html   # последняя итерация
        └── production.html            # текущая продакшен-страница
```

## Как запустить локально

Любой статический сервер подойдёт. Проще всего:

```bash
npm install       # ставит `serve` (только один раз)
PORT=3000 npm start
# откройте http://localhost:3000
```

Или без Node — например `python3 -m http.server 3000`, затем тот же адрес.

Файлы полностью автономны: `roboforex-organic-analytics/index.html` можно
открыть и просто двойным кликом (`file://…`) — весь датасет лежит прямо в
странице, интернет не нужен. Единственное внешнее обращение во всех страницах —
шрифт Google Fonts; без него всё работает, просто применяется системный шрифт.

## Проекты

### RoboForex — органический трафик
Интерактивный дашборд помесячной аналитики органики (сессии из BigQuery),
период **2022-01 → 2026-07**. Показывает, какие страны, разделы, страницы и
хосты выросли или упали и когда, плюс слой Google Search Console (клики, показы,
позиции, CTR) для диагностики причин. Как пользоваться — см. блок ниже.

### Compound Calculator
Четыре живых направления редизайна калькулятора сложного процента плюс текущая
продакшен-страница. Каждый концепт — самостоятельный интерактивный прототип.

## Как работать с отчётом RoboForex

Отчёт — одна страница с девятью вкладками. Сверху — **глобальная панель периода**
(пресеты и перетаскиваемая шкала-«кисть»): выбранный диапазон применяется сразу
ко всем вкладкам.

| Вкладка | Что показывает |
|---|---|
| **Обзор** | Итоговая динамика сессий, пики/провалы, топ движений за период |
| **Динамика и тепловая карта** | Помесячная тепловая карта по любому разрезу (абс./доля/индекс) |
| **События и эффект** | Что изменилось вокруг конкретного месяца — эффект правок/удалений |
| **Сравнение периодов** | Два диапазона рядом, кто вырос и кто упал |
| **Гео × раздел** | Куб «страна × раздел сайта» |
| **Оценка возврата** | Контрфактическая оценка потолка возврата снятых разделов |
| **Страницы** | Таблица топ-страниц с разбивкой по странам |
| **Search Console** | Слой GSC: клики/показы/позиция/CTR помесячно + разбор потери март→июнь 2026 |
| **Методика и находки** | Как считалось, ограничения данных, ключевые выводы |

**Разрезы** (где есть селектор «Разрез»): страна, раздел сайта, каталог URL,
страница, языковая папка, тематический сегмент, хост/поддомен, устройство,
поисковая система, ОС.

**Режимы значений:** абсолют, доля (% ко всему трафику за месяц), индекс к
базовому месяцу. Почти в каждой таблице есть поиск по маске URL/названию и
выгрузка в CSV.

> **Важно про данные.** Личный кабинет (`my.roboforex.com` и родственные
> `my.roboforex.pro/.org`, `my28.roboforex.org`) и веб-терминал
> `webtrader.roboforex.com` **исключены** из отчёта — это не органика привлечения,
> а трафик уже залогиненных пользователей. Поэтому общий итог здесь ниже, чем
> «весь сайт целиком».
